Dropdown select matching the input system, with a chevron affordance.

```jsx
<Select label="Sensor type" defaultValue="gas">
  <option value="gas">Gas detector</option>
  <option value="temp">Temperature probe</option>
  <option value="vib">Vibration sensor</option>
</Select>
```

40px tall, 8px radius feel via input border. Focus blue ring; supports `error`.
