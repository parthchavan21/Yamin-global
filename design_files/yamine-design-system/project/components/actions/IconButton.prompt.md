Icon-only button for toolbars and compact actions; requires an accessible `label`.

```jsx
<IconButton label="Settings" variant="outline"><GearIcon/></IconButton>
<IconButton label="Add device" variant="brand"><PlusIcon/></IconButton>
```

Variants: `ghost` (default), `outline`, `brand`. Sizes `sm` (32px) and `md` (40px). Use Lucide icons at 18–20px inside.
